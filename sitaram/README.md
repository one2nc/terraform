# terraform-practice

```
a) - Write terraform script to create a multi-AZ setup(Mumbai region).
    Their should be:
     - Internet Gateway.
     - NAT.
     - Application Loadbalancer.
     - two bastion machines (one per az).
     - two service machines(one per az).
     - one rds(my-sql/postgres).
    The service boxes, rds should be in private subnet.
    Use minimum size of the box.
b) - Use any configuration manager ansible/chef to provision a LAMP stack.
````
